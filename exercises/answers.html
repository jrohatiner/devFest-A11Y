Exercise 2

  • Missing Form Labels: The input fields for "First Name" and "Last Name" do not have associated label elements, which are crucial for screen reader users.
  • Non-Standard Form Controls: The "Submit" button is implemented using a div element instead of a "button" or "input type="submit". This can cause issues with screen reader and keyboard navigation.
  • Missing Alternative Text for Images: The img tag for the kitten image lacks an alt attribute, which is necessary to convey the content and function of the image to screen reader users.
